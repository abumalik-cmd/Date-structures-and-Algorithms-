# Array

Коллекция решений задач на массивы на Java.

## Файлы

- **Concatenation-Array.java** - Создает массив, который является конкатенацией исходного массива с самим собой (удваивает массив).

- **findDisappearedNumbers.java** - Находит все числа от 1 до n, которые отсутствуют в массиве. Использует маркировку отрицательными числами.

- **findErrorNums.java** - Находит дубликат и пропущенное число в массиве от 1 до n.

- **findMaxConsecutiveOnes.java** - Находит максимальное количество последовательных единиц в бинарном массиве.

- **shuffle.java** - Перемешивает массив из 2n элементов: [x1,x2,...,xn,y1,y2,...,yn] в [x1,y1,x2,y2,...,xn,yn].

- **smallerNumbersThanCurrent.java** - Для каждого элемента массива подсчитывает, сколько чисел в массиве меньше него.
